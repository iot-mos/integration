Real-time sensor data streamed over MQTT via IoT-MOS. Subscribe to a topic and the broker pushes a JSON reading whenever a sensor event occurs. Use this channel for live use cases (floor maps, occupancy displays, alerts). For historical or aggregated data, use the HTTP REST API instead.
Servers
secure-mqtt://{endpoint}
production · MQTT over TLS

MQTT broker over TLS. The host is your account-specific endpoint, provided by IoT-MOS during onboarding.

endpoint — Provided by IoT-MOS.
Authorize x509 (mutual TLS)

Mutual TLS using an X.509 client certificate issued by IoT-MOS and bound to your companyId. There is no token to paste — authentication is performed by the certificate presented during the TLS handshake.

sensorData One sensor's live readings, streamed as they happen.
SUBSCRIBE iotmos-api/v1/{env}/{companyId}/{buildingId}/{roomId}/{boothId} Live readings for one sensor ▾

Subscribe with MQTT wildcards to widen scope: use + for a single level (all sensors/booths in a room) or # for everything below a level (all sensors/booths in a company or building).

Quick start Python · paho-mqtt

Install the MQTT client

pip install paho-mqtt

Connect over TLS with your X.509 client certificate and subscribe. Use + to match a single topic level (e.g. all booths in a room) or # to match everything below a level (e.g. all sensors in your company):

import json
import ssl
import paho.mqtt.client as mqtt

ENDPOINT    = "your-endpoint.iot-mos.example"  # provided by IoT-MOS during onboarding
PORT        = 8883                             # MQTT over TLS (secure-mqtt)
COMPANY_ID  = "your-company-id"

# X.509 client certificate (mutual TLS), issued by IoT-MOS during onboarding
CA_CERT     = "iotmos-ca.pem"
CLIENT_CERT = "client.crt"
CLIENT_KEY  = "client.key"

# Topic: iotmos-api/v1/{env}/{companyId}/{buildingId}/{roomId}/{boothId}
# Subscribe to every sensor in your company, live environment:
TOPIC = f"iotmos-api/v1/live/{COMPANY_ID}/#"

def on_connect(client, userdata, flags, reason_code, properties):
    if reason_code == 0:
        print(f"Connected. Subscribing to {TOPIC}")
        client.subscribe(TOPIC, qos=1)
    else:
        print(f"Connection refused: {reason_code}")

def on_message(client, userdata, msg):
    reading = json.loads(msg.payload)
    print(f"[{msg.topic}] {reading['sensorid']} ({reading['type']}): {reading['metrics']}")

client = mqtt.Client(mqtt.CallbackAPIVersion.VERSION2, protocol=mqtt.MQTTv5)
client.tls_set(
    ca_certs=CA_CERT,
    certfile=CLIENT_CERT,
    keyfile=CLIENT_KEY,
    tls_version=ssl.PROTOCOL_TLS_CLIENT,
)
client.on_connect = on_connect
client.on_message = on_message

client.connect(ENDPOINT, PORT, keepalive=60)
client.loop_forever()   # blocks and reconnects automatically
