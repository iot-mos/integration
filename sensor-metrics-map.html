IoT-MOS Sensor Metric Map REFERENCE

The metrics object in a sensor reading varies by sensor type. This page lists which metric fields appear for each sensor, with the JSON key and the value you can expect. Derived from com.daisen.enums.SensorType / MetricType. JSON keys are the lowercased output names used in the published payload.
Value encoding

Every metric value is transmitted as a JSON string. Numeric metrics are numbers in string form (e.g. "2.96") and are shown below as Numeric (string); state metrics are string literals (e.g. "OPEN").

Always present

In addition to the type-specific fields below, every metrics object also includes dbm (radio signal strength, in dBm), and every reading carries a timestamp.
